# AirChef

AirChef is a platform to buy and sell delicious home-cooked meals, wherever you are in the world.
##### Current Features
  - Full user authentication via [Stormpath]
  - Once logged in, users can view a list of all the meals on the explore page
    - Meals are populated via API call to our [server]
    - Pulling down on the list will refresh it
    - Tapping on a meal entry brings up further details
  - Users can access a form to submit their own meal
    - Meal posting not yet supported


   [stormpath]: <https://stormpath.com/>
   [server]: <https://github.com/marstankev/AirChefAPI>
